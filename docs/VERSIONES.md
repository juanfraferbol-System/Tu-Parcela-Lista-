# Versiones — Tu Parcela Lista

## Convención

`MAJOR.MINOR.PATCH`

- MAJOR: cambios grandes de plataforma.
- MINOR: módulos nuevos.
- PATCH: correcciones o mejoras internas.

## Versiones objetivo

- 2.0.x: arquitectura, limpieza, estabilidad.
- 2.1.x: Supabase y publicación de parcelas.
- 2.2.x: Mercado Pago y planes.
- 2.3.x: CRM.
- 3.0.x: IA, recomendador y automatizaciones.
