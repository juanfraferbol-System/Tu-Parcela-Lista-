# Cambios aplicados - Tu Parcela Lista

## Archivos incluidos
- `index.html`: limpieza del `<head>`, eliminación de duplicaciones, enlace a “Cómo comprar” y barra simple de pasos.
- `style.css`: mejoras visuales unificadas, sidebar más protagonista, botones premium y mejor responsive.
- `como-comprar.html`: página del proceso de compra incluida desde la mejora anterior.
- `_redirects`: rutas recomendadas para Netlify.
- `netlify.toml`: configuración base para publicar y cachear recursos.
- `404.html`: página de error personalizada.

## Importante
Esto ordena la parte visible y reduce errores de estructura. Para resolver completamente CRM real, login seguro, estadísticas reales, envío automático de PDF por WhatsApp y guardado de interesados, se necesita una segunda etapa con backend: Supabase, Firebase o Netlify Functions.

## Recomendación de instalación
Reemplazar `index.html` y `style.css` en la raíz del proyecto. Agregar los archivos nuevos en la raíz. Mantener tus carpetas `image/`, `agente/` y scripts actuales.
