# Tu Parcela Lista V3 - Producción

Versión preparada para subir a Netlify sin carpeta `.git` anidada.

## Cómo subir a Netlify
1. Entra a Netlify.
2. Add new site → Deploy manually.
3. Arrastra el ZIP o esta carpeta completa.
4. El archivo `index.html` debe quedar en la raíz del deploy.

## Archivos agregados para producción
- `robots.txt`
- `sitemap.xml`
- `manifest.webmanifest`
- `politica-privacidad.html`
- `terminos.html`
- `css/v3-production.css`

## Importante CRM
El CRM actual es una versión local/demo basada en navegador. Para producción real con seguridad, conectar a Supabase Auth + base de datos.

## Google Ads
Antes de activar campañas verifica:
- Dominio con HTTPS.
- WhatsApp funcionando.
- Política de privacidad visible.
- Términos visibles.
- Search Console instalado.
- Google Analytics 4 / Google Tag Manager configurado.
