# TPL v0.11 — Inicio rápido + Responsive Premium

## Nuevas mejoras
- Se agregó página `tecnologia.html` para explicar el uso de Inteligencia Artificial, incluyendo ChatGPT de OpenAI, sin dar a entender asociación oficial.
- Se agregó acceso visible en footer a Tecnología e IA.
- Se reforzó el Home con tres acciones directas: comprar parcela, publicar parcela y cotizar casa.

## Responsive móvil
- Mejoras adicionales para menú móvil, botones táctiles, formularios y páginas de publicación.
- Ajustes para evitar desbordes horizontales y mejorar la experiencia en teléfonos.

## Comercial
- Se mantiene el flujo de captación de dueños, corredores y empresas con estado pendiente de revisión.
- El panel local de publicaciones queda más ordenado y preparado para Supabase.
